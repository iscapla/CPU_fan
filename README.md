# CPU_fan
Code to implement a CPU cooler for the Raspberry Pi 3

Installation
------------
No extra packages are needed to run this program.

Possible configuration
----------------------
File: 

	/home/pi/Programs/config.ini

Content:
	
	[Fan]
	waitTime_sec: 5
	avg_list_num: 4
	pin: 19
	minTMP: 44
	maxTMP: 48
	[Others]
	verbose: 0
